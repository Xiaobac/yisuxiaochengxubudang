### 阶段二：后端 API 开发（2-3天）

**Day 3-4: 核心 API 实现**

- [ ] 用户认证 API
  - POST /api/auth/register - 注册
  - POST /api/auth/login - 登录
- [ ] JWT 中间件实现
- [ ] 角色权限检查中间件
- [ ] 酒店 CRUD API
  - GET /api/hotels - 列表查询（支持筛选）
  - GET /api/hotels/:id - 详情
  - POST /api/hotels - 创建（商户）
  - PUT /api/hotels/:id - 更新（商户）
  - DELETE /api/hotels/:id - 删除（商户）
- [ ] 图片上传 API
  - POST /api/upload - 图片上传（multer）
- [ ] 审核 API
  - GET /api/reviews/pending - 待审核列表（管理员）
  - POST /api/reviews - 审核酒店（管理员）
  - PUT /api/hotels/:id/status - 更新状态（管理员）

**测试：**
- 使用 Postman 测试所有接口
- 验证权限控制是否生效

**预期成果：**
- 12 个 API 接口全部实现
- Postman 测试通过
