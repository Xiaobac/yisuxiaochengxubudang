更新了所有酒店的图片 URL，将失效的 Unsplash 图片链接替换为本地上传的图片。

```sql
UPDATE hotels 
SET images = '["http://localhost:3000/uploads/1770189062477-9-2026-02-03185959.png"]'
WHERE images LIKE '%unsplash%';
```
